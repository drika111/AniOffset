# AniOffset
Blender Addon - Propagate pose transformation offsets across selected keyframes in the active Action Slot. Designed for use with rigged characters in Blender.

---


-Eng-

Hi everyone,

I’ve developed a new add-on for Blender 5.1 (and newer) that lets you adjust multiple keyframes directly from the viewport. I’ve tested it thoroughly across different scenes and computers. The add-on is called Anim Offset Pro, because it helps you propagate pose adjustments across several keyframes at once. ✅

Here’s how to use it after installation:

Select the rig and switch to Pose Mode;
In the side panel, choose the Action you want to modify from the dropdown menu;
Select the corresponding Action Slot (the add-on will show its users for clarity);
Define the frame range where the offset should be applied by entering the start and end frames;
Click the “Offset” button. From now on, any transformation you make to the pose in the viewport will be automatically applied across the selected frame range! ✨
Important: If you want to undo a change made while the offset is active, press Ctrl+Z twice — once to undo your edit and once more to remove the offset propagation.

I hope someone liked it and have fun using it! ^^~

-Pt (Br)-

Oi gente,

Eu desenvolvi um novo add-on para Blender 5.1 (e versões mais recentes) que permite ajustar múltiplos keyframes diretamente pela viewport. Testei bastante em cenas e computadores diferentes. O nome é Anim Offset Pro, porque ele ajuda a propagar ajustes de pose por vários quadros da animação. ✅

Veja como usar depois de instalar o add-on:

1. Selecione o rig e entre no modo Pose;
2. No painel lateral, selecione a Ação (Action) que você deseja modificar no menu suspenso;
3. Escolha o Slot da ação correspondente (o add-on mostra qual é o usuário do slot, para garantir clareza);
4. Defina o intervalo de quadros onde o offset será aplicado, preenchendo os campos de frame inicial e final;
5. Clique no botão “Offset”. A partir daí, qualquer transformação feita na pose pela viewport será aplicada automaticamente em todos os quadros do intervalo escolhido! ✨


Importante: Se for desfazer uma alteração feita com o offset ligado, aperte Ctrl+Z duas vezes — uma para desfazer a edição e outra para remover a propagação do offset.


Espero que alguém goste e se divirta usando ele! ^^~
