# AniOffset
Blender Addon - Propagate pose transformation offsets across selected keyframes in the active Action Slot. Designed for use with rigged characters in Blender.

---


-Eng-

Hi everyone,

I’ve developed a new add-on for Blender 5.1 that lets you adjust multiple keyframes directly from the viewport. I’ve tested it thoroughly across different scenes and computers. The add-on is called Ani Offset, because it helps you propagate pose adjustments across several keyframes at once. ✅

Here’s how to use it after installation:

1. Enable the addon in the side panel named Ani Offset;
2. In the side panel, choose the Action you want to modify from the dropdown menu;
3. Define the frame range where the offset should be applied by entering the start and end frames;
4. In pose mode, select the bones you're going to change and click in "capture pose"
5. Adjust your pose and with all adjusted bones selected click on "propagate pose;

Done! All the changes were applied into that range! Now the pose of this bones/rig is adjusted but the animation curves all were maintained ^~ ✨

Important: If you want to undo a change made while the offset is active, press Ctrl+Z twice — once to undo your edit and once more to remove the offset propagation.
I hope someone liked it and have fun using it! ^^~

-Pt (Br)-

Oi gente,

Eu desenvolvi um novo add-on para Blender 5.1 (e versões mais recentes) que permite ajustar múltiplos keyframes diretamente pela viewport. Testei bastante em cenas e computadores diferentes. O nome é Anim Offset Pro, porque ele ajuda a propagar ajustes de pose por vários quadros da animação. ✅

Veja como usar depois de instalar o add-on:

1. Ligue o addon no painel lateral chamado Ani Offset;
2. Selecione a "action" e "slot"  que você deseja modificar no menu suspenso;
3. Defina o intervalo de quadros onde o offset será aplicado, preenchendo os campos de frame inicial e final;
4. No modo pose, selecione os ossos que vai modificar e clique em "capture pose" (capturar pose);
5. Ajuste sua pose e com todos os ossos que ajustou selecionados clique em "propagate pose";

Todas as mudanças foram aplicadas nesse intervalo! Agora o posicionamento desses ossos/rig está ajustado   mas as curvas de animação dele foram mantidas ^~ ✨


Importante: Se for desfazer uma alteração feita com o offset ligado, aperte Ctrl+Z duas vezes — uma para desfazer a edição e outra para remover a propagação do offset.


Espero que alguém goste e se divirta usando ele! ^^~
